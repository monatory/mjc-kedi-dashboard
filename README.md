## 📚 함께 보기

### 메인 대시보드
- **광역시·도 카드 클릭** → 자치구·진출 학과·전공/비전공 비율 펼침
- **기업유형 펼치기** → 대표 기업·대표 학과
- **📚 분석 방법론 펼치기** → 데이터 출처·분류 기준·추정 방식·한계·향후 보강 방향
- **상위 30 학과 차트** 옆 ⚠ **표본 부족 학과 10곳** 카드 → 통계 신뢰도 낮은 학과 별도 안내

### 학과 카드 (그리드 하단)
- 🟢 **초록 테두리**: 취업률 ≥60% (우수 학과)
- ⚪ **파란 테두리**: 평균 (50~60%)
- 🔴 **빨강 테두리**: <50% (지원 필요)
- ⚠ **주황 표시**: 대상자 10명 이하 (통계 신뢰도 낮음)
- 카드에 마우스 올리면 **"🔍 상세 분석 보기 →"** 표시 — 클릭 시 학과별 8개 상세 분석으로 이동

### 학과별 상세 페이지
- **KPI 6장**: 취업률·유지·진학·평균급여·평균학점·자격증
- **📑 보조 분석 (3열 카드 그리드)**: 직무 대·소분류, 기업 유형, 산업 분류, 근무 지역, 자격증, 급여, 직장 유지율
- **🎯 전공/비전공 분석** (펼치기): 전공 직무·전공 기업처 도넛 + 회사 목록
- **🏛 취업 기업 (상위 20개)**: 학과 졸업생이 실제 진출한 회사·인원
- **📊 취업 구분별 비중**: 건강보험 직장가입자·프리랜서·1인 창업 등

## ⚠ 데이터 한계 (요약)

- KEDI 자격증 통계는 큐넷(국가기술자격) 기반 — 면허증·민간자격 미포함
- 시·군·구 단위 데이터는 서울만 제공
- 일부 지표는 학과 가중합 추정 (개인 단위 매칭 부재)
- 표본 작은 학과 10개 (대상자 ≤10명) — 비율 해석 시 주의
